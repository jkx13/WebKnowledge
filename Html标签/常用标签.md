## font标签
```
<font color="red" face="宋体" size="7">文本</font>
```

## 常用字符实体
|	符号	|	描述	|	实体名称	|
|--	|--	|--	|
|	<	|	小于号	|	`&lt;`	|
|	>	|	大于号	|	`&gt;`	|
|       |    空格    |  `&nbsp;` |

## 超链接
```html
//当前页面打开
<a href="https://www.baidu.com" target="_self"></a>
//新窗口打开
<a href="https://www.baidu.com" target="_blank"></a>
```

## ul与ol标签
```html
//type设置列表符号
<ul type="none">
	<li></li>
</ul>
<ol>
	<li></li>
</ol>
```
